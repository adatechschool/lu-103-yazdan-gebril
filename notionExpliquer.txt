Les structures de données sont des moyens spécifiques d'organiser et de stocker des données afin qu'elles puissent être consultées et travaillées de manière efficace. 
Les structures de données définissent la relation entre les données et les opérations pouvant êtres effectués dessus .

Le choix d'un structure de données dépend du problème que nous devons résoudre et du type de données dont nous disposons. Une structure de données inappropriée peut entrainer des durées de fonctionnement(inutilement longues), des pertes de mémoire et un gaspillage de stockage. Ce qui peut raisonnablement entraîner, une perte de plusieurs millions d'euros, voir de quelques jours quand on parle de données à l'échelle.

Les structure de données retrouver en python : 
 
- Tuple 
- Str
- Range 
- List 
- Dict
- Set
